# Animatch
Проект за SoftUni - ASP.NET Fundamentals


Animatch е уеб платформа, която свързва собственици на домашни любимци. Идеята е хората да могат да намират приятели за игра или разходка за своите животни, като публикуват обяви и се свързват помежду си.

Основни функции:
Списък с всички животни и филтриране по име, категория и град.
Добавяне на нова обява за животно (име, вид, порода, описание, град, телефон, снимка).
Детайли за конкретно животно.
Редактиране и изтриване на обява – само от собственика, който я е добавил.
Потребителска регистрация и логин чрез ASP.NET Identity.
След успешен логин/регистрация – пренасочване към страницата с животни.

Използвани технологии:
ASP.NET Core MVC (Razor Views)
Entity Framework Core (Code First)
ASP.NET Identity (логин/регистрация и потребители)
SQL Server (база данни)
Bootstrap (стилизация)

Инструкции за ползване:
Приложението се стартирва с home страницата където е обобщена основната идеа на проекта.
Сред това потребителят трябва да се регистрира за да използва приложението и да добавя обяви на животни.


Проблеми при разработка:
Накрая най-големият проблем беше с миграциите и промяна на типа на PhoneNumber
Първо PhoneNumber беше int, после го смених на string.
Базата още беше с int, а кодът очакваше string, затова се получи грешка InvalidCastException.
Имаше и конфликтен default constraint, който блокираше промяната на колоната.
Решението беше да се махне constraint-а и да се промени типът в базата, или да се направи нова миграция, която да актуализира схемата правилно.
